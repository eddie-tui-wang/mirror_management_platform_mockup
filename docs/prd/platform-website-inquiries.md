【二期】# Platform > Website Inquiries

**路径**: `/dashboard/website-inquiries`
**适用门户**: Platform
**访问权限**: `platform:website_inquiries:view`
**菜单位置**: 顶级菜单 > Website Inquiries

---

## 一、页面概述

Website Inquiries 页面供平台管理员查看官网表单提交记录，即潜在客户通过官网渠道发起的询价或咨询信息。

---

## 二、页面内容

### 2.1 询价列表

| 列名 | 说明 |
|------|------|
| Time | 表单提交时间 |
| Name | 提交者姓名 |
| Company | 提交者所在公司名称 |
| Email | 提交者联系邮箱 |
| Role | 提交者职位/角色 |
| Venue | 提交者描述的使用场景/门店类型 |

### 2.2 搜索

- 支持按姓名（Name）、公司（Company）或邮箱（Email）进行模糊搜索。
- 搜索防抖 500ms，也可按 Enter 立即执行。
- 搜索框支持清空。

### 2.3 分页

- 默认每页显示 10 条记录。

---

## 三、操作

本页面为只读视图，无操作入口。

---

## 四、黑盒规则

### R-WI-01：表单数据由官网接收后写入系统
Website Inquiry 记录由官网表单（公开页面）提交后通过后端写入，非系统内部操作产生。平台管理员只能查看这些线索，不可编辑或删除。

### R-WI-02：询价记录不关联任何平台账号
Website Inquiry 是独立于账号体系之外的线索记录，提交者的邮箱不与任何 User 账号关联。销售跟进后，如决定开通账号，需通过"创建渠道"或"创建客户"流程单独操作。
